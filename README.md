# Contrast

### Note

Create a `master`-branch and adjust `_config.yml` if you want to use this theme in the root folder of `yourname.github.io`.

### Features

 - responsive
 - minimal theme
 - dark theme
 - pagination
 - archive page
 - feed.xml generation
 - syntax highlighting
 - comments ([disqus](https://disqus.com/) / [isso](http://posativ.org/isso/))
